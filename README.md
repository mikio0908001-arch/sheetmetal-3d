# sheetmetal-3d — 板金 図面→概形3D（ブラウザ完結）

2D機械図面（板金プレート）から、作業者が**その場で形を確認できる概形3D**を作るブラウザアプリ。
ローカルインストール不要・全デバイス対応・**図面データは端末内（ブラウザ）だけで処理し、サーバへ送信しない**。

> 見た目確認用の概形であり、寸法保証はしない。実STEPが客先承認を通ったら差し替える運用（ハイブリッド）。

## 使い方（作業者）
1. 公開URL（GitHub Pages）を開く（PC/スマホ/タブレット可）。
2. 図面（PNG/JPG/TIF）を読み込む（ドラッグ＆ドロップ可）。
3. **① スケール**：既知寸法（例 220 の寸法線）の両端を2点クリック → 実寸 mm を入力。
4. **② 輪郭**：板の角を順にクリック。
5. **③ 穴**：穴の中心をクリック（直径を入力）。
6. **板厚**を入れて **3D生成 ▶**。右画面をドラッグで回転。
7. 必要なら **GLB保存** でモデルをダウンロード。

※ TIF/TIFF は**ブラウザ内で自動的にPNG相当へ変換**して読み込む（UTIF.js 使用・サーバ送信なし）。複数ページTIFは1ページ目を表示。ただし CCITT G4 など一部の圧縮は未対応のことがあり、その場合は PNG/JPG に変換して読み込む。

## 開発（クラウドファースト / Codespaces）
1. このリポを **Code → Codespaces → Create codespace** で起動（ブラウザのVS Code）。
2. ターミナルで `python3 -m http.server 8000` → 転送ポート8000をプレビュー。
3. `index.html` を編集 → 保存 → プレビュー更新。
4. `main` に push すると **GitHub Pages が自動で再ビルド・公開**（Deploy from a branch 方式）。

### 初回だけの設定
- リポの **Settings → Pages → Build and deployment → Source = Deploy from a branch → Branch: `main` / `/(root)`** にする（1回だけ）。
- 以後は `main` への push で自動公開。公開URLは `https://mikio0908001-arch.github.io/sheetmetal-3d/`（Settings→Pages にも表示）。

## 設計メモ
- 3D生成は **three.js の ExtrudeGeometry**（穴あき多角形の押し出し）で**完全にクライアント側**。Python/CADカーネル不要。
- 機密：手動なぞり経路は図面がブラウザ内に留まりサーバへ送信しない＝クラウド配信と機密保持が両立。
- 次の予定：曲げ（フランジ）対応、複数パネル（溶接）、図番キーのモデル保存、（任意）Claudeによる寸法読み補助。
